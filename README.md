# TNS SandBox Animated Landing Page

[Live Demo](https://tns-sandbox-ui.vercel.app/) | [GitHub Repo](https://github.com/Ahmad-Munaam/TNS-SandBox-UI)

## About

A project assigned by a company called TNS. They wanted an animated, modern landing page for their digital agency brand. I built the whole UI from scratch with heavy focus on animations and smooth interactions.

This was a game changer for my UI career. The company was genuinely impressed with the final result — and that opened more opportunities for me.

## Built With

- React.js
- Tailwind CSS
- JavaScript
- Vite

## Features

- Hero section with animated text and call-to-action
- Services section (Web Design, Graphic Design, 3D Animation)
- Working strategy steps with numbered sections
- Company facts stats (projects completed, happy customers, awards)
- Team members section
- Latest projects portfolio grid
- Customer testimonials with avatars
- FAQ accordion section
- Fully responsive — animations work smoothly on all devices

## What I Learned

- How to make a company notice my work — this project proved that my UI skills could attract real clients
- Animations need to feel natural, not overdone. Too much motion is annoying; just enough makes the site feel premium
- Working on a company-assigned project is different from personal practice — there are real expectations and deadlines
- How to structure a complete multi-section landing page that looks professional from top to bottom
